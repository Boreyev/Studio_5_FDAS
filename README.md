# How to get app running:
1. open vs code terminal
2. create virtual env: 
``` 
python -m venv venv 
```
4. activate virtual env: 
``` 
venv/bin/activate 
```
6. install numpy:
``` 
pip install numpy 
```
8. install face recognition:
``` 
pip install face_recognition 
```
10. install opencv: 
``` 
pip install opencv-python
```
